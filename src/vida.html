Sorry, but Vida is a client side application, and requires a modern web browser that can support WebGL. Here is a list of cross platform browsers that should be able to run Vida

  • Mozilla's Firefox
  • Google's Chromium or Chrome if you prefer compiled binaries
  • Opera
